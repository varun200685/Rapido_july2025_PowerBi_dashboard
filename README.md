# Rapido_july2025_PowerBi_dashboard

Rapido Operations Dashboard (Power BI)

Overview

An interactive Power BI dashboard analyzing Rapido ride-booking operations, built from a real bookings dataset. The dashboard tracks key operational and revenue metrics to support business decision-making around fleet performance, booking trends, and vehicle-type utilization.

Objective

To transform raw ride-booking data into actionable operational insights — covering booking volume, revenue, cancellations, and vehicle-type performance — for stakeholders in a ride-hailing / mobility business context.

Dataset


Source: Rapido bookings dataset (real-world ride-booking data)
Granularity: Individual booking-level records
Key fields: Booking ID, date/time, vehicle type, fare/revenue, pickup/drop details, booking status (completed/cancelled), driver/customer info (as available)
